# Report Styling Guidelines  
Ross Property — A Division of Ross Creative

## Tone & Style
- Clear  
- Professional  
- Neutral  
- Direct  
- No jargon unless required  
- No filler language  

## Structure
All reports follow this order:

1. Title Page  
2. Summary  
3. Key Findings  
4. Detailed Sections  
5. Conclusion  
6. Notes / Limitations  

## Formatting
- Headings: H1 for titles, H2 for sections, H3 for subsections  
- Lists: Use bullet points for clarity  
- Paragraphs: Short, concise, single idea per paragraph  
- Numbers: Use consistent units (sq ft / sq m)  
- Dates: Use UK format (DD/MM/YYYY)  

## Visuals
- Diagrams optional  
- No decorative elements  
- Only functional visuals  

## Language Rules
- Avoid subjective language  
- Avoid speculation  
- Use advisory phrasing: “Based on available information…”  
- Use consistent terminology across all reports  

## AI Integration
Reports may be drafted or supported by the Documentation Assistant (Scribe).  
All outputs must be reviewed before delivery.

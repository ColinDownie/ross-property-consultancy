# Internal Workflow — Ross Property

## Overview
This workflow ensures consistent delivery across all services.

---

## 1. Intake
- Client submits intake form  
- Client Experience Assistant logs details  
- Initial review by consultant  

## 2. Scoping
- Clarify objectives  
- Identify required service  
- Determine deliverables  
- Prepare proposal  

## 3. Instruction
- Client accepts proposal  
- Project is opened  
- Workflow assigned  

## 4. Analysis
Depending on service:
- Valuation  
- Feasibility  
- Site Analysis  
- Due Diligence  
- Advisory  

AI Assistants support calculations, documentation, and compliance checks.

## 5. Documentation
- Draft report created  
- Reviewed internally  
- Amended where required  

## 6. Delivery
- Final report sent to client  
- Clarifications handled  
- Project closed  

## 7. Archiving
- Documents stored  
- Notes logged  
- Data retained securely

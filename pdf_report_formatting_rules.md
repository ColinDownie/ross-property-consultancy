# PDF‑Ready Report Formatting Rules  
Ross Property — A Division of Ross Creative

## Page Layout
- A4 portrait  
- 20–25mm margins  
- Single column  
- No decorative elements  

## Typography
- Typeface: Inter  
- Body: 14–16px  
- Headings: 20–32px  
- Weight: Regular / Medium / Bold  
- Colour: Slate Black (#1A1A1A)  

## Structure
1. Title Page  
2. Summary  
3. Key Findings  
4. Detailed Sections  
5. Conclusion  
6. Notes / Limitations  

## Spacing
- 1.4 line height  
- 16–24px spacing between sections  
- Consistent paragraph spacing  

## Visuals
- Simple diagrams only  
- No gradients  
- No decorative icons  

## Branding
- Logo in top‑left of title page  
- No additional branding elements  

## File Output
- Export as PDF  
- No compression  
- No watermarks

IT Framework

The IT Framework defines the systems, infrastructure, security, and support model that underpin the organisation’s digital operations. It is designed for reliability, security, and scalability.

Scope

  • Core systems and platforms
  • Access control and security
  • Data management and backup
  • Support, incident handling, and change

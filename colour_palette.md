# Colour Palette — Ross Property

## Primary Palette
### Slate Black
Hex: #1A1A1A  
Usage: Text, headings, primary elements

### Concrete Grey
Hex: #E5E5E5  
Usage: Backgrounds, dividers, neutral surfaces

### Structural White
Hex: #FFFFFF  
Usage: Primary background, clean space

---

## Secondary Palette
### Blueprint Blue
Hex: #2A4E7A  
Usage: Highlights, diagrams, emphasis

### Surveyor Green
Hex: #4A6F5A  
Usage: Environmental or site‑related visuals

---

## Usage Rules
- Use Slate Black for all text  
- Use Structural White as the default background  
- Use Concrete Grey sparingly for separation  
- Use Blueprint Blue only for emphasis  
- Use Surveyor Green only for environmental diagrams  

No gradients.  
No decorative colour usage.
